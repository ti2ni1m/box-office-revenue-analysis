# Box Office Revenue Analysis

This project analyzes a dataset of 3000 movies to uncover trends in box office revenue, budget, popularity, and release patterns.

## Project Overview

The goal of this analysis is to:

Identify high-performing movies by revenue
Explore the relationship between budget and revenue
Analyze trends in movie releases over time
Understand factors influencing popularity
## Dataset

The dataset includes:

Budget and revenue
Release date information
Popularity scores
Runtime and other metadata
## Data Cleaning
Removed columns with excessive missing values or complex structures
Converted release dates into datetime format
Extracted:
Year
Month
Day
Weekday
## Key Insights
* Highest Revenue Movie: The Avengers (~$1.5B)
* Highest Budget Movie: Pirates of the Caribbean: On Stranger Tides
* Most movies are between 1.5–2 hours long
* Most movies are released on Fridays
* Peak release years were around 2013–2015
* Popularity distribution is highly skewed (few very popular movies)
